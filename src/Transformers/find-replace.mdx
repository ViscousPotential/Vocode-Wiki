# Find & Replace

A simple way to find and replace text

## Config

| Name       |   Type    | Default | Description                                |
| :--------- | :-------: | :-----: | :----------------------------------------- |
| find | `string` | `<none>` | The substring to find in the transcriber result |
| replace | `string` | `<none>` | The string to replace the `find` substring with in the transcriber result |

## Examples
- find: "hashtag "  replace: "#"
- find: "buy "  replace: "#buy"
- find: "get "  replace: "#buy"
